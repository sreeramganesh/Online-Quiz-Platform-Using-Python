Online Quiz Platform

Description

The Online Quiz Platform is a Python-based application that conducts multiple-choice quizzes, checks answers, and calculates scores in real time. It demonstrates core programming concepts such as conditional logic, data structures, and user input handling.

Features
 * Multiple-choice quiz system
 * Real-time answer validation
 * Automatic score calculation
 * Simple and user-friendly interaction
 * Easy to modify and extend with new questions

Technologies Used
  Python

How to Run the Project 
 1.Install Python on your system.
 2.Download or clone this repository.
 3.Open the project folder.
 4.Run the Python file using:
        Quiz_platform.py 
 5.Answer the questions displayed on the screen.
 6.View your final score at the end of the quiz.
